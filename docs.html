Shapeshifter

Shapeshifter helps you design responsive websites and interfaces by providing a constantly available set of viewport controls within your browser, which allows you to test the responsiveness of your creations at every step of the way.

View on GitHub

Keyboard Shortcuts

  • alt-shift-u: Browser viewport
  • alt-shift-i: iPhone 3G/4 viewport
  • alt-shift-o: iPhone 5 viewport
  • alt-shift-p: iPad viewport
  • alt-shift-l: Rotate viewport (only available for mobile viewports)
  • alt-shift-g: Go to a new location
